Easily manage the visibility of widgets.

== Description ==

With Widgets Master you can easily manage the visibility of Widgets by Categories/Taxonomies, Pages, Post types and more.

== Installation ==

1. Upload the folder `widgets-master` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Appearance -> Widgets to configure your widgets
